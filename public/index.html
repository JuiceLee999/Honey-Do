Features Guide
🔍
Search

Type in the search bar to find projects by name, description, notes, category, contractor, or property.

🎯
Filters & Sort

Use the toolbar buttons to filter by status or priority. Sort by due date, cost, priority, or last modified.

📋
Subtasks

Add a checklist of steps to any project in the detail panel. Drag to reorder them.

🔨
Contractors

Save contractor contacts and assign them to projects. Manage all contractors via the Contractors button.

👥
Sharing

Share individual projects with other Honey-Do users (view or edit access) from the project's detail panel.

⬇
PDF Export

Export any project as a printable PDF using the PDF button in the panel footer.

🌙
Dark Mode

Toggle dark mode with the moon/sun button in the header. Your preference is saved automatically.
